# 🧠 Sentiment Analysis App

This is a real-time sentiment analysis web app built using HuggingFace Transformers and Streamlit.

## 🚀 Features
- Positive / Negative sentiment prediction
- Confidence score
- Emoji-based output
- Local AI model deployment

## 🛠 Tech Stack
- Python
- Streamlit
- HuggingFace Transformers
- PyTorch

## ▶ How to Run
```bash
streamlit run app.py
